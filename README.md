## 📝 Description
This task focuses on creating a high-end user experience for contact interfaces. It leverages **Bootstrap 5** for structure and **Vanilla JavaScript** for robust client-side validation. The design is inspired by modern dashboard aesthetics, featuring a deep dark theme, custom-styled inputs, and instant feedback loops to guide the user through the submission process.

## ✨ Features
* **Real-time Validation:** Instant error/success feedback for name, email, and subject fields using Regex and logic.
* **Smart Character Counter:** Dynamic counter for the message field that changes color (Warning/Danger) as you approach the limit.
* **Advanced Dark UI:** Custom CSS implementation with glassmorphism touches and smooth `slideUp` entry animations.
* **Fully Responsive:** Perfectly optimized for mobile, tablet, and desktop using Bootstrap's grid system.
* **Interactive Submission:** Includes a loading state (spinner) on the submit button and automated scrolling to errors.
* **Vanilla JavaScript:** Fast, lightweight DOM manipulation without the need for external libraries.

## 🛠️ Technologies Used
* **HTML5 & CSS3** (Custom properties, Grid/Flexbox, and Keyframe animations)
* **Bootstrap 5** (Layout and responsive utility classes)
* **Vanilla JavaScript** (Form events and validation logic)
* **Font Awesome 7.0.1** (Professional iconography)
* **Google Fonts** (Syne & DM Sans)
